### 模块化开发
- CommonJS
Node最先提出js模块化的概念，Node.js服务器端通过exports，module.exports来输出模块。并使用requires同步载入模块


- AMD
异步加载模块，主要用于浏览器端，主要应用有requireJS


- es6 module
es6定义了js的模块化方案，取而代之的是前面一系列规范，一统江湖。鉴于es6的支持率，需要用到bable转义
